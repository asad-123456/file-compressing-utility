This simple Python tool provides a quick way to compress and decompress text files using Run-Length Encoding (RLE). It features a basic Tkinter-based interface where you can pick a file, see the compression ratio, and automatically sort your results into dedicated "compressed" and "decompressed" folders. Since RLE works by grouping identical characters together, it's perfect for files with lots of repetition, though the app will warn you if the file isn't a good candidate for this type of compression. It even handles tricky symbols like commas and pipes by escaping them to ensure your data stays intact during the round trip.
